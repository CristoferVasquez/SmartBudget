# SmartBudget

### Despliegue configuracion smartbudget para usar pages
se despliega codigo completo.
### Detalles e informacion
Se uso sass para trabajo de estilos con una metodologia BEM para lo semantico y con SMACSS para la estructura, esto lo preferi por el orden y capacidad de variedad que permiten los estilos de manera independiente al trabajarse, ademas se agregaron dos modales para implementar funcionalidad de ingreso de valores, tanto de Ingresos como egresos de dinero, se implemento grafico usando financeChart generando vistas tanto para cuando esta en pantallas grandes como en pantallas chicas.
## Importante

para trabjar codigo de scss o sass se debe usar npm run sass, el script esta definido en el .json de node.js
